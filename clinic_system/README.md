## 🚀 تشغيل على VS Code — خطوة بخطوة

### الخطوة 1: تثبيت Python
تأكد إن Python 3.8+ مثبت:
```
python --version
```
لو مش موجود: https://www.python.org/downloads/

### الخطوة 2: افتح المجلد في VS Code
```
File → Open Folder → اختر مجلد clinic_system
```

### الخطوة 3: افتح Terminal في VS Code
```
Terminal → New Terminal  (أو Ctrl + `)
```

### الخطوة 4: ثبّت المكتبات
```bash
pip install flask pandas openpyxl
```

### الخطوة 5: شغّل السيستم
```bash
python app.py
```

### الخطوة 6: افتح في المتصفح
```
العياده (الدكتور):   http://localhost:5000/doctor
شئون الطلاب:         http://localhost:5000/affairs
```

## 🔗 ربط العياده بشئون الطلاب — على الشبكة الداخلية

### لو العياده وشئون الطلاب على نفس الجهاز:
- العياده:    http://localhost:5000/doctor
- شئون:       http://localhost:5000/affairs

### لو على أجهزة مختلفة في نفس الشبكة (Intranet):

**الخطوة 1:** على الجهاز اللي فيه السيستم، عدّل آخر سطر في app.py:
```python
# قبل:
app.run(debug=True, port=5000)

# بعد:
app.run(host='0.0.0.0', port=5000, debug=False)
```

**الخطوة 2:** اعرف الـ IP بتاع الجهاز:
```bash
# Windows:
ipconfig
# مثال النتيجة: IPv4 Address: 192.168.1.10

# Mac/Linux:
ifconfig
```

**الخطوة 3:** من أي جهاز في الشبكة افتح:
```
العياده:     http://192.168.1.10:5000/doctor
شئون الطلاب: http://192.168.1.10:5000/affairs
```

> ✅ دي اللينكات اللي هتديها للدكتور وشئون الطلاب

## 🔄 تحديث ملفات الطلاب والموظفين

فقط استبدل الملفات في مجلد `data/`:
- `data/students.xlsx` — يحتوي على أعمدة: `م`, `الكود`, `الاسم`, `البرنامج`
- `data/employees.xlsx` — يحتوي على أعمدة: `م`, `رقم الملف`, `الاسم`

بعدين أعد تشغيل السيستم:
```bash
python app.py
```

## 📊 قاعدة البيانات

السيستم بيحفظ تلقائياً في `clinic.db`:
- `visits` — كل زيارات العياده
- `affairs_reports` — تقارير شئون الطلاب
- `medicines` — مخزون الأدوية
